### 如何编译

编译之前先确保你的设备上拥有 AndroidSDK 与 JDK，需要`clone`此项目你还需要一个`git`

然后运行以下代码

```sh
git clone https://github.com/CCZU-OSSA/cczu-vpn-android.git
cd cczu-vpn-android
./gradlew assembleDebug
```
